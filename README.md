# 電気情報工学実験第２ ハンドジェスチャーゲームの審判を行うアプリ

## 作品紹介
- 小学生の時に行った、手を使ったゲームの審判を行うアプリを実装した(寿司じゃんけん）
- mediapipeを使用して、手の形を認識しルールにのっとり判定する
- 動きに合わせて表示される演出も作成した

## 環境構築
### 1.リポジトリのクローン
git clone <リポジトリのURL>

### 2. 仮想環境構築
ルート直下で以下のコマンドを入力するとカレントディレクトリに.venv/フォルダが作成される
```
python -m venv .venv
```

### 3.仮想環境の有効化
macOS/Linux
```
source .venv/bin/activate

```
WIndows(PowerShell)
```
.venv\Scripts\Activate.ps1
```

### 4. pythonファイルの実行
実行したいディレクトリへ移動し（```cd <ディレクトリ名＞```）コマンドを入力
```
python <ファイル名>
```

### 5.voicevoxのダウンロード＆起動(任意)
ずんだもんの掛け声を流すためにvoicevoxをダウンロードして起動する
