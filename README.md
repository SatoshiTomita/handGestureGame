# ハンドジェスチャーゲームの審判を行うアプリ

## 環境構築
### 1.リポジトリのクローン
git clone <リポジトリのURL>

### 2. 仮想環境構築
ルート直下で以下のコマンドを入力するとカレントディレクトリに.venv/フォルダが作成される
```
python -m venv .venv
```

### 3.仮想環境の有効化
macOS/Linux
```
source .venv/bin/activate

```
WIndows(PowerShell)
```
.venv\Scripts\Activate.ps1
```

### 4. pythonファイルの実行
実行したいディレクトリへ移動し（```cd <ディレクトリ名＞```）コマンドを入力
```
python <ファイル名>
```
